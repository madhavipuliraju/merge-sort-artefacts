1 Introduction

This document builds the split step interactive artefact(HTML).

2 Features of the artefact

  • Artefact provides demo for split step.
  • User can see a randomly generated array of size 10 in the beginning. Size of the array is fixed.
  • User can click the Reset button to generate a new array at any time.
  • User can click on Next Step button to manually view the demo at his/her own speed.
  • User can click on Previous Step button to undo a step.

3 HTML Framework of Simple Bubble Sort

3.1 Head Section Elements

Title, meta tags and all the links related to Js, CSS of this artefact comes under this section. 

  <title>Bubble Sort</title>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="../css/mergesort.css">
  <script src="../js/split.js"></script>

3.2 Body Section Elements

3.2.1 Instruction box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction">
  <b>Instructions</b>: 
  <ul>
  <li>Click on the Next Step button to see how the Split Tree progresses to build. </li>
  <li>Notice the indices where the arrays are getting split. Also keep an eye out for Odd and Even splits.</li>
  </ul>
  </div>

3.2.2 Canvas

This is an canvas, which is used for graphics. 

    <div>
      <canvas id="canvas" width="400" height="240"></canvas>
    </div>

3.2.3 Controller Elements

This are the controller buttons(Next Step, Reset, Previous Step) of an artefacts used for performing/demonstrating the artefact. 

<div id="buttondiv" >
  <center>
      <button type="button" class = "button" id="next">Next Step</button>
      <button type="button" class = "button" id="previous">Previous Step</button>
      <button type="button" class = "button" id="reset">Reset</button>
  </center>
</div>
